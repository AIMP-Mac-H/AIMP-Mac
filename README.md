## 💡 Overview

<a href="#aIMP">AIMP</a> serves audio enthusiasts, audiophiles, and power users who want more control over their music playback than Spotify, Apple Music, or simple media players provide — particularly users with large local music libraries in diverse formats who want precise equalization, gapless playback, and DSP control.

The <a href="#aIMP">local music library use case</a> is AIMP's primary value — users who maintain collections of FLAC albums, lossless rips, and high-quality audio files in diverse formats want a player that handles all formats natively, plays them without quality degradation, and provides the equalization and DSP tools to optimize listening for specific hardware. The <a href="#aIMP">audiophile playback use case</a> uses AIMP's bit-perfect output mode, high-quality audio processing chain, and per-device output configuration for critical listening through quality headphone amplifiers and DACs.

The <a href="#aIMP">free pricing model</a> makes AIMP attractive as a Winamp and foobar2000 alternative — delivering comparable feature depth to paid players and the most feature-rich free alternatives without any cost. <a href="#aIMP">Windows heritage with Mac version</a> — AIMP originated as a Windows application with a devoted power-user following, and the Mac version brings the same feature set to macOS.</p>

<a href="#aIMP">Tag editor</a> views and edits audio file metadata — title, artist, album, year, genre, album art, and other ID3 and similar metadata tags directly within AIMP without requiring a separate tag editing application. <a href="#aIMP">Audio converter built-in</a> converts audio files between supported formats from within the player — batch conversion of a library portion without launching separate conversion software.

<a href="#aIMP">Internet radio support</a> plays streaming internet radio stations from saved stream URLs or M3U playlists — integrating streaming radio into the same player used for local library playback. <a href="#aIMP">Visualization plugins</a> display audio-reactive visual effects during music playback for the visual component of music listening that some users prefer alongside high-quality audio.</p>

Gapless playback is the audio feature that most clearly separates serious music players from simple ones. When an album is designed to flow continuously between tracks — a live recording, a classical symphony, a concept album — the brief silence that standard track-sequential playback inserts between files breaks the intended listening experience. AIMP's gapless mode eliminates that interruption, allowing albums to play as their creators intended.

## 🙋 FAQ

<details>
<summary>Can AIMP edit audio file tags?</summary>

Yes. A built-in tag editor views and edits audio file metadata including title, artist, album, and album art.

</details>

<details>
<summary>Does AIMP support visualization plugins?</summary>

Yes. Audio-reactive visualization plugins display during music playback.

</details>

<details>
<summary>Can I customize AIMP's appearance?</summary>

Yes. AIMP supports skins and visual themes from a community library alongside built-in appearance options.

</details>

<details>
<summary>Does AIMP support DSP effects?</summary>

Yes. DSP effects including tempo, pitch, reverb, and stereo enhancement apply in real time during playback.

</details>

<details>
<summary>Is AIMP available on platforms other than Mac?</summary>

Yes. AIMP is primarily a Windows application with a Mac version. The Windows version has a larger plugin ecosystem.

</details>
